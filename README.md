# Service-Net — Sitio Web Corporativo

Sitio web institucional para **Service-Net**, empresa de soluciones informáticas. Desarrollado como proyecto freelance con enfoque en claridad de servicios, SEO básico y estructura multipágina.

## Tecnologías utilizadas

- HTML5 semántico
- CSS3 — layout responsivo, diseño por secciones
- JavaScript vanilla
- `sitemap.xml` y `robots.txt` para SEO básico
- `_headers` para configuración de seguridad HTTP

---

## Decisiones técnicas

**Estructura Simple** Se optó por una single-page.

**SEO básico incluido.** El proyecto incluye `sitemap.xml` para indexación por crawlers y `robots.txt` para control de acceso. Configuración de headers HTTP de seguridad vía `_headers`.

**Sin dependencias externas.** El sitio corre con HTML, CSS y JS puros, sin frameworks ni librerías, lo que garantiza tiempos de carga mínimos.
